# Images Folder

## Snake Bite Image

Please save your snake bite first aid image as:
- **Filename**: `snakebite-guide.jpg` or `snakebite-guide.png`
- **Location**: This folder (`project/public/images/`)

The image will automatically appear at the end of the Snake Bite guidance panel.

### Image Details
- **Shows**: 8-step first aid guide for snake bite
- **Displays**: Only when user clicks on "Snake Bite" emergency card
- **Position**: After "Seek Emergency Help If" section
- **Styling**: Rounded corners, shadow, responsive width

### Supported Formats
- JPG/JPEG
- PNG
- WebP

### Recommended Size
- Width: 800-1200px
- Height: Auto (maintains aspect ratio)
- File size: < 500KB for optimal loading
